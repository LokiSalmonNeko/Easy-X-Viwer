TwitterAPI.io API 金鑰

如何取得 API 金鑰

  1. 訪問 TwitterAPI.io 官方網站
  2. 註冊帳號並取得 API 金鑰
  3. 將金鑰貼到下方輸入框並儲存

設定 API 金鑰後，您可以在主頁面選擇使用 TwitterAPI.io 來顯示影片（適用於年齡限制貼文或官方 embed 無法顯示的情況）

金鑰將安全地儲存在本地配置檔案中

使用說明

1. 設定 API 金鑰

在上方輸入您的 TwitterAPI.io API 金鑰並儲存

2. 在主頁面選擇 API

新增貼文時，選擇「TwitterAPI.io」作為載入方式

3. 適用場景

  • 年齡限制貼文
  • 官方 embed 無法顯示的貼文
  • 需要更穩定載入的貼文

注意：TwitterAPI.io 是付費服務，使用時會產生費用。請參考 官方定價。
